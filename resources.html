Resources for Learning CSS Grid

There are tons of resources on the internet for learning CSS Grid. Some are better than others depending on your learning style. I myself am largely a visual learner. Here are the resources I found most useful when learning CSS Grid.

Rachel Andrews

Grid By Example

Rachel Andrews is a web developer, speaker and writer from the U.K. Rachel has an entire website dedicated to teaching CSS Grid called Grid By Example.

Grid by Example has a ton of great documentation and articles, written by Rachel, that explain CSS Grid in a clear and instructive manner. She also has video tutorials where she explains different CSS Grid techniques and properties as she codes. I found these videos to be highly informative and easy to follow along.

Rachel also speaks at many conferences concerning web design and technology. Her presentations are definitely worth checking out if you're interesting in hearing her perspective on web design and advocacy for CSS Grid.

Frontend Conference Zurich.

Jen Simmons

Labs.JenSimmons

Jen Simmons is a graphic designer, web developer, speaker and developer advocate at Mozilla. She has a multitude of skills but is known for her expertise in HTML and CSS web standards, as well as her advocacy for CSS Grid.

Labs.JenSimmons is her website which contains links to various topics of her expertise like CSS Grid, Web Design, and Firefox Nightly.

Her youTube channel, Layout Land, is a great channel to learn what she has to say and teach concerning CSS Grid.

She has a bunch of conference talks concerning the future of web design which I highly recommend. Many of her talks center around web standards and how they have transformed over the years to meet the needs of designers.

MDN Web Docs - Basic Concepts of Grid Layout

Basic Concepts of Grid Layout

The MDN Web Doc's documentation on CSS Grid is an excellent resource.

Honestly, I find most developer documentation to be arduous but they have a great basic concepts guide to learning CSS Grid.
